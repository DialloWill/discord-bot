# 🤖 Discord Automation Bot

A bot to automate tasks and manage community interactions on Discord.

## ⚙️ Features
- Welcome messages for new users
- Moderation (kick, ban, mute)
- Fun commands (jokes, quotes, polls)

## 🔧 Tech
- Python
- discord.py
